CSS լեզվի թեստեր:

Գիտելիքների ստուգատես

  
Ծրագրի Բացատրությունը

 
  1. Ինչպես են անվանում տառատեսակային խմբերը:
    1.   font_style
    2.   font_family
    3.   font_size
    4.   font_variant
  2. Normal, bold, bolder, lighter պարամետրերը որ խմբին են պատկանում:
    1.   font_weight
    2.   font_family
    3.   font_size
    4.   font_variant
  3. Որ ոճն է ,որ ունի ավելի կլորացված և լայն տառերով տառատեսակ:
    1.   serif
    2.   sans-serif
    3.   fantasy
    4.   cursiv
  4. Ինչ է իրագործում Monospace ոճը:
    1.   տառերի եզրերին գեղարվեստական ձևավորում
    2.   տալիս է բոլոր տառերին միևնույն լայնությունը
    3.   տալիս է բոլոր տառերին հաստություն
    4.   տալիս է բոլոր տառերին թեքություն
  5. Ինչպես է իրագործվում արտաքին ներբեռնված CSS ֆայլը:
    1.   Ճիշտ պատասխանը բացակայում է
    2.   փաստաթղթում հայտարարել CSS ներդրված ոճ
    3.   փաստաթղթի էլեմենտում հայտարարել CSS գծային ոճ
    4.   փաստաթղթի վերնագրային մասում իրականացնել արտաքին ֆայլի կցում
